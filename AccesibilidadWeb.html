INVESTIGACIÓN 3: ACCESIBILIDAD WEB

¿Qué es la Accesibilidad Web?

La accesibilidad web es la práctica de diseñar y desarrollar sitios web para que todas las personas puedan usarlos con sin problema, independientemente de sus capacidades o situación. La accesibilidad web permite combinar la programación y el diseño para lograr un mayor entendimiento y un fácil manejo de cualquier sitio web accesible,

Importancia

  • Inclusividad: Diseñar para la diversidad promueve inclusión para diversas personas con sus propias capacidades y limitaciones.
  • Cumplimiento legal: Muchos países y organizaciones adoptan la accesibilidad Web como requisito legal, exigiendo que el contenido sea fácil de usar por cualquier usuario.
  • Mejor experiencia del usuario: Las páginas web accesibles no solo sirve para personas con capacidades limitadas, si no que támbien mejora la calidad de experiencia de los usuarios promedio al ser más limpias y sencillas las páginas web.
  • Optimización de motores de búsqueda (SEO): La accesibilidad web támbien potencia al SEO, ajustando las preferencias de los motores de búsquedas por ser un sitio web fácil de manejar.

Principios de las WCAG

Los principios de Accesibilidad al Contenido Web (WCAG), han sido desarrolladas por el Consorcio World Wide Web, y son pautas que nos facilitan la accesibilidad de los sitios web a todas las personas que posean capacidades limitadas, incluso a los usuarios promedio le hes de utilidad, teniendo las organizaciones que quieran cumplir la Ley de Estadounidenses con Discapacidades (ADA), seguir las pautas mencionadas anteriormente. Actualmente hay 4 principios WCAG:

  • Perceptible: El contenido debe de ser visible para las personas que posean mayor dependencia visual, y ciertos problemas en alguno de sus otros sentidos.
  • Operable: Los sitios web deben de ser facilmente manejables, ya sea mediante comandos de voz, o usando únicamente el teclado, en caso de no poseer un ratón.
  • Comprensible: Básicamente se debe entender de forma clara y sencilla el contenido del sitio web.
  • Robusto El sitio web debe de poder transmitir la información y el contenido a todos los usuarios posibles, eso incluye a usuarios que utilizan técnologia de asistencia.
