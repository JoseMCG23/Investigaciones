INVESTIGACIÓN 1: SEO (OPTIMIZACIÓN PARA MOTORES DE BÚSQUEDA) Y RENDIMIENTO

¿Qué es el SEO?

El SEO (Optimización para motores de búsqueda) se utiliza para poder ayudar a los buscadores y usuarios en internet para que puedan entender mejor el contenido que crees en tu sitio web. en otras palabras, sirve para ayudarte a que puedan encontrar tu página y tomar la desición de poder visitarla en el buscador.

SEO y Velocidad de Carga

Las Métricas web principales son un conjunto de métricas que sirven para medir en tiempo real la experiencia de los usuarios basandose en la velocidad de carga del contenido, la interacción y la estabilidad visual de las páginas web. Es por esto que se le recomiendan a los creadores de sitios web tener buenas Métricas web principales para conseguir buenos resultados a la hora de realizar búsquedas, para asi ser recompensados con buenos posicionamientos principales. Las Métricas web Principales son:

  • Renderizado del mayor elemento con contenido (LCP): Se encarga de medir la velocidad de carga del contenido para otorgar buena experiencia de usuario. El valor de LCP recomendado es que sea inferior a 2,5 segundos.
  • Interaction to Next Paint (INP): Se encarga de medir la capacidad de respuesta para proporcionar una buena experiencia de usuario. Se recomienda evitar un valor de INP superior a 200 milisegundos.
  • Cambios de diseño acumulados (CLS): Se encarga de medir la estabilidad visual para ofrecer una buena experiencia de usuario. Se recomienda un valor inferior a 0,1.

SEO Técnico

El SEO técnico se le considera como la base fundamental de cualquier estrategia de posicionamiento web. Sin una estructura bien optimizada, contenido de calidad no sería capaz de alcanzar su máximo potencial. Según la guía de Google LLC sobre datos estructurados, que es donde se esta basando para realizar esta investigación, “añadir structured data puede habilitar resultados enriquecidos en la búsqueda y fomentar una mayor tasa de clics” (CTR).

Metaetiquetas semánticas

Se consideran como metaetiquetas semánticas a elementos como: <title>, <meta name= "description"> y <meta name="robots"> , y sirven para informar a los motores de búsqueda acerca del tema del que se esta tratando en la página web y poder ayudar a la hora de que los usuarios realicen búsquedas relacionadas al contenido de tu sitio web.

Datos estructurados

Un buen uso correcto de la jerarquía de los encabezados (<h1>, <h2>, <h3>, etc.) facilitan a los motores de búsqueda el poder comprender dicha jerarquía de información permitiendo un mejor uso del SEO. Si una página web no posee una buena estructura, esto podría dificultar el registro del contenido.

Optimización de imágenes

Esto no solo mejora la experiencia del usuario, si no que también mejora el SEO: Los archivos bien nombrados, los atributos alt bien descriptivos, y los formatos actuales como WebP sirven para que los motores de búsqueda puedan comprender también el contenido visual y lo registren correctamente. Un ejemplo es que Google nos recomienda usar nombres de archivos bien descriptivos y diferenciables en lugar de usar los nombres por defecto o genéricos, algo como "IMG00023.JPG".

Etiquetas Open Graph y Twitter Cards

Las etiquetas Open Graph son fragmentos de código HTML que nos permiten controlar el cómo se muestra el contenido de una página al ser compartida en redes sociales, como Facebook, Whatsapp o Instagram. Estas etiquetas nos ayudan a definir el título, la descripción, la imagen y el link, o URL que se visualizarán en la vista previa del enlace, mejorando la apariencia y despertando el interés de los usuarios para que quieran visitar el sitio web.
Por otro lado, las Twitter Cards funcionan de forma similar, permitiendo a los sitios web controlar el cómo se muestra su contenido al ser compartido en la plataforma de Twitter. Estas etiquetas utilizan un formato muy parecido al de Open Graph e incluiyen elementos como el título, la descripción, el autor y una imagen destacada.
